# Procesador de Nómina Electrónica México

**Obligatorio:**
- CFDI Nómina 1.2 + complementos vigentes.
- Cálculo correcto de percepciones y deducciones.

**Flujo:**
1. Recibe datos de empleados.
2. Calcula nómina completa.
3. Genera XML por empleado.
